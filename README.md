# 🌱 AI-Driven Site-Specific Weed Management (SSWM)

An AI-powered precision agriculture platform that uses **Deep Learning** and **Computer Vision** to detect weed species from field images and generate intelligent zone-wise herbicide treatment recommendations. The system helps farmers optimize herbicide usage, reduce costs, and minimize environmental impact through precision farming techniques.

---

## 📖 Overview

Traditional weed management involves spraying herbicides uniformly across an entire field, resulting in excessive chemical usage and higher costs. This project leverages a **ResNet-50 Convolutional Neural Network (CNN)** to identify weed species accurately and provide targeted herbicide recommendations for affected areas.

---

## 🚀 Features

- 🌿 AI-based weed detection using **ResNet-50 CNN**
- 🎯 Achieves **94.7% classification accuracy**
- ⚡ Fast prediction with **23 ms inference time**
- 📍 GPS-based field mapping
- 🔥 Weed density heatmap visualization
- 💊 Zone-wise herbicide treatment recommendations
- 💰 Cost analysis for optimized weed management
- 📄 PDF report generation
- 🌐 Responsive web application

## 🎯 Workflow

1. Upload a field image.
2. The ResNet-50 model detects weed species.
3. The system identifies weed-affected zones.
4. GPS mapping visualizes weed locations.
5. A heatmap is generated.
6. Herbicide recommendations are created.
7. Cost analysis is performed.
8. A PDF report is generated.
